About Me

Benvenuti nel mio mondo digitale! Sono Andrea Ruzittu, un appassionato Full Stack Developer in erba, con una formazione solida e un'innata curiosità per il mondo della tecnologia. La mia avventura nel campo dello sviluppo software è iniziata con un corso intensivo presso Epicode, dove ho trascorso sei mesi immerso in 950 ore di formazione. Durante questo periodo, ho esplorato e acquisito competenze in una varietà di tecnologie, tra cui HTML, CSS3, JavaScript, SASS, Bootstrap, TypeScript, Angular, OOP, Java, SQL, JPA, JMP, Git e SpringBoot. Prima di tuffarmi in questo affascinante viaggio, ho conseguito una laurea in Economia e Management. Questo background mi ha fornito una prospettiva unica, permettendomi di comprendere non solo il "come", ma anche il "perché" dietro le soluzioni di sviluppo software, allineando tecnologia e strategia aziendale. Sono un individuo determinato, che non teme le sfide e che è sempre alla ricerca di nuove opportunità per crescere e migliorare. Nel mio percorso, ho imparato l'importanza del lavoro di squadra, della comunicazione efficace e dell'approccio hands-on nei progetti. Nel mio portfolio, troverete esempi del mio lavoro e progetti ai quali ho contribuito, che riflettono la mia passione per la creazione di soluzioni software efficaci e la mia capacità di adattarmi a diversi ambienti e requisiti. Grazie per aver visitato il mio sito. Spero che il mio lavoro vi ispiri e non vedo l'ora di collaborare con voi per trasformare le idee in realtà digitali. Cordiali saluti, Andrea Ruzittu

Projects

Easy Reservation

Easy Reservation è una applicaizone che si occupa di gestire prenotazioni per strutture alberghiere, essa contiene anche la possibilità di accedere come amministratori in modo da poter creare, modificare ed eliminare le stanze, hotel e le prenotazioni le tecnologie utilizzate per il back-end sono le seguenti: Java (v17): Linguaggio di programmazione utilizzato. Spring Data JPA: Per l'interazione con il database. Spring Security: Per gestire l'autenticazione e la sicurezza. PostgreSQL: Database utilizzato. Lombok: Utilizzato per ridurre il codice boilerplate. JWT (JSON Web Tokens) (v0.11.5): Per l'autenticazione stateless. Java Validation API (v2.0.1.Final): Per la validazione delle email. Spring Boot Devtools: Per funzionalità di sviluppo come il "hot reload". Per quanto riguarda il front-end le tecnologie utilizzate sono le seguenti: Angular: Framework di sviluppo front-end. TypeScript: Linguaggio di programmazione utilizzato con Angular. SCSS: Preprocessore CSS per una gestione più avanzata dello stile. Bootstrap: Framework CSS per la creazione di interfacce utente moderne e responsive. Per visionare il progetto accedete al link e seguite le istruzioni sul readme.
